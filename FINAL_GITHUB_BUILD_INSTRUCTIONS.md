# JE Final GitHub Build

This project is complete and ready for GitHub Actions.

## Required repository setup

Your GitHub repo must contain either:

- this project extracted at repo root (`app/`, `gradlew`, `settings.gradle.kts` visible), or
- this whole project uploaded as a single `.zip` file at repo root.

The workflow at `.github/workflows/main.yml` supports both.

## Build

Go to:

Actions → Build JE APK → Run workflow
